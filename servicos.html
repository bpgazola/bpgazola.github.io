Impressos

Produção gráfica de qualquer tipo de impresso: cartões de visita, folders, panfletos, catálogos, cartazes, revistas, envelopes, cartas, documentos, livros e muito mais. É só enviar o conteúdo que tomamos conta de todo o processo; desde definir o formato ideal a escolha de imagens ilustrativas e elementos gráficos. No final do projeto enviamos para você os arquivos finais abertos em formato vetorial prontos para a impressão.

Identidade

Não importa se você precisa de um projeto rápido ou apenas um logotipo. Pensamos em uma estruturação completa que pode incluir criação tipográfica, iconografia, criação de banners e planos de fundo para redes sociais, criação de wallpapers, design de cartão de visitas, diagramação de documentos e outros impressos. Todos os projetos acompanham guia de aplicação da marca com estruturação de cores, ícones para internet, variações para diferentes aplicações, exemplos de uso. Tudo isto, é claro, entregue em arquivos de alta resolução otimizados para cada tipo de aplicação.

Desenvolvimento Web

Semântica, acessibilidade, performance e tecnologia. Esta é a fórmula secreta para desenvolver sites bem estruturados, fáceis de navegar e que carregam rápido. Sites desenvolvidos utilizando linguagens modernas como HTML5, CSS3 e JavaScript. Se você precisa de um sistema de administração podemos oferecer do versátil WordPress.

Design de interface

Projetamos interfaces para web, sistemas e aplicativos móveis, utilizando guias de estilo para organizar todos os elementos do projeto. A atenção aos detalhes está presente em todos os layouts com foco no usuários e aplicações intuitivas com design efetivo. Além de layouts exclusivos também atualizamos seu design com novos projetos de arquiteturas de informação. Projetos de re-design de sites institucionais, portais, lojas virtuais, blogs e muito mais.
